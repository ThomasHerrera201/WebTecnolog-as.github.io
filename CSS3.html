CSS3

CSS3 es el lenguaje encargado del diseño y la presentación visual de una página web. Mientras HTML define el contenido, CSS se ocupa de cómo ese contenido se ve.

HITOS IMPORTANTES DEL CSS3

Las Media Queries permitieron adaptar el diseño de una página a diferentes tamaños de pantalla, mejorando la experiencia en celulares y tablets.

CSS3 incorporó animaciones, transiciones y transformaciones que permiten crear efectos visuales sin usar JavaScript.

Se añadieron características como bordes redondeados, sombras y gradientes, lo que facilitó la creación de interfaces modernas.

Con CSS3 se puede

- Cambiar colores, tipografías y tamaños de texto.

- Definir márgenes, bordes y espacios entre elementos.

- Crear diseños organizados y ordenados.

- Adaptar la página a diferentes tamaños de pantalla (diseño responsivo).

- Añadir animaciones y efectos visuales.

CSS3 permite separar el contenido del diseño, lo que facilita el mantenimiento y la mejora de los sitios web. Gracias a CSS, una misma página puede verse diferente según el dispositivo o las preferencias del usuario.
