HTML5

HTML5 es el lenguaje de marcado utilizado para construir la estructura y el contenido de una página web. No es un lenguaje de programación, sino un lenguaje que indica al navegador qué elementos existen en la página y cómo están organizados.

HITOS IMPORTANTES DEL HTML5

Se incorporaron etiquetas como header, nav, section, article y footer, que mejoraron la organización del contenido y facilitaron la accesibilidad y el posicionamiento en buscadores.

HTML5 permitió reproducir contenido multimedia con las etiquetas audio y video sin necesidad de complementos externos como Flash.

HTML5 fue diseñado para funcionar correctamente en distintos navegadores y dispositivos, impulsando el desarrollo de sitios web adaptables.

Con HTML5 se puede

- Crear títulos, párrafos y listas

- Insertar imágenes, videos y audios.

- Añadir enlaces para navegar entre páginas.

- Construir formularios para recopilar información del usuario

- Una de las principales mejoras de HTML5 es el uso de etiquetas semánticas, que permiten organizar mejor el contenido y hacerlo más comprensible tanto para las personas como para los motores de búsqueda. Además, HTML5 está diseñado para funcionar correctamente en distintos dispositivos, como celulares y tablets.
