Teconologías Básicas

El desarrollo web se basa en un conjunto de tecnologías fundamentales que permiten crear páginas y aplicaciones accesibles desde un navegador. Entre ellas, las más importantes y básicas son HTML5, CSS3 y JavaScript. Estas tres tecnologías trabajan de forma conjunta y cumplen funciones diferentes, pero complementarias, para que un sitio web funcione correctamente.

Estas se basan en:

HTML5

CSS3

JavaScript
